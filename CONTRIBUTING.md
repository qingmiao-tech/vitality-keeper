# 贡献指南

感谢你愿意参与元气守恒·筑基令。这个项目还在早期阶段，任何真实使用反馈、平台验证和小而清晰的修复都很有价值。

## 开发环境

```bash
npm install
npm run dev
```

Windows 可使用：

```bat
dev.bat
```

## 提交建议

- 保持改动聚焦，一次 pull request 解决一个明确问题。
- 涉及 UI 文案时，请同时检查中文和英文。
- 涉及 Rust 后端、窗口、托盘或系统集成时，请说明验证过的平台。
- 不要提交 `node_modules/`、`src-tauri/target/` 或安装包产物。

## 常见贡献方向

- 修复 Windows 多屏和托盘边界问题。
- 验证 macOS / Linux 构建链路。
- 改进无障碍、键盘操作和小屏布局。
- 补充更多语言翻译。
