# v0.1.9 版本更新说明

元气守恒·筑基令 v0.1.9 聚焦于“多语言体验更完整、系统设置更清晰、快捷键链路更稳定”。

## Highlights

- 新增阿拉伯语界面支持，并补齐 RTL 从右到左布局适配。
- 优化语言选择器，所有语言选项始终以各自母语显示，不再受当前界面语言影响。
- 重构系统设置页的信息分组，让“启动与系统 / 语言与文案 / 更新”三组内容更清晰。
- 完善全局热键链路，支持录制、持久化、运行时状态反馈与冲突提示。
- 将应用版本与安装包版本统一提升至 `v0.1.9`。

## Downloads

- `vitality-keeper_0.1.9_x64-setup.exe`
- `vitality-keeper_0.1.9_x64_en-US.msi`

## Release Notes

- 本版本继续沿用 GitHub Releases 与 `latest.json` 作为自动更新分发方式。
- 如果自动更新签名链路需要完整产物，打包环境仍需配置 `TAURI_SIGNING_PRIVATE_KEY`。
- 阿拉伯语支持覆盖控制中心、休息页以及系统托盘菜单文案，并保持与“平衡 / 归元”两套文案风格兼容。

## English Summary

Vitality Keeper v0.1.9 adds Arabic UI support with RTL layout handling, improves the language picker so each option always renders in its native script, refines the system settings information architecture, and strengthens the editable global shortcut flow.
