# v0.1.6 开源首发说明

元气守恒·筑基令 v0.1.6 是 Tauri 重构版的首次开源发布。

## Highlights

- Rust + Tauri 重构，后台占用更轻。
- 支持本地视频引导休息，可选择单文件或文件夹。
- 支持多显示器休息窗口同步。
- 支持视频时长结束或固定休息时长两种策略。
- 支持全屏铺满和完整显示两种视频呈现方式。
- 支持系统托盘、推迟限制、开机自启和中英文界面。

## Downloads

- `vitality-keeper_0.1.6_x64-setup.exe`
- `vitality-keeper_0.1.6_x64_en-US.msi`

## Known Notes

- 当前预构建安装包优先支持 Windows x64。
- macOS 和 Linux 可从源码构建，安装包仍在验证中。
- 全局快捷键和 DND 仍会继续加强。

## English Summary

Vitality Keeper v0.1.6 is the first open-source release of the Tauri rewrite. It provides local video guided breaks, multi-display synchronization, tray controls, launch-at-startup support, and bilingual UI. Windows x64 installers are provided first; macOS and Linux can be built from source.
