# v0.1.8 版本更新说明

元气守恒·筑基令 v0.1.8 聚焦于“快捷键可用、文案可切换、控制中心更清晰”。

## Highlights

- 接入真实的全局快捷键注册，`Ctrl + Alt + Q` 现在可以直接重置休息周期。
- 控制中心新增热键状态徽标与“立即重置”动作，便于确认快捷键是否注册成功。
- 新增文案风格设置，默认保持公开版的平衡表达，同时提供更有道家气息的“归元”风格。
- 休息页与控制中心共享同一套文案风格配置，避免切换后前后页面语气不一致。
- 优化系统设置页样式，让语言切换、文案风格和热键状态在深浅主题下都更协调。

## Release Notes

- 本版本继续沿用 GitHub Releases + `latest.json` 作为自动更新源。
- 如果 `Ctrl + Alt + Q` 显示注册失败，通常表示被系统中其他应用占用了同一快捷键。
- 默认文案风格仍为更通用的公开表达，切换为“归元”后会同步影响控制中心与休息页。

## English Summary

Vitality Keeper v0.1.8 adds a real global reset shortcut for `Ctrl + Alt + Q`, a selectable copy-style layer with an optional Dao-inspired “Ritual” voice, and a cleaner control-center presentation for shortcut status and system settings.
