# 产品文案与体验审阅

## 当前定位

元气守恒·筑基令适合作为“轻量桌面健康休息提醒 + 本地视频引导”工具推广。它的差异点不是单纯计时，而是把本地视频、全屏休息、多显示器同步和托盘控制组合成完整的休息流程。

## 文案策略

- 保留“筑基令”作为品牌记忆点。
- 功能说明使用直白表达，例如“本地视频引导”“休息节律”“系统启动后自动在后台运行”。
- 英文文案不直译修真隐喻，优先表达为 wellness、break reminder、local video guidance。
- 避免过度私人化或过度风格化的表达，让第一次使用的人也能理解每个设置项。

## 体验建议

- README 首屏应先告诉用户它解决什么问题，再展示“筑基令”的品牌气质。
- 控制中心适合继续保持东方视觉风格，但设置项要像工具一样清楚。
- 休息页需要让倒计时、进度线和视频状态始终可见，避免用户误以为卡住。
- 托盘菜单需要在非休息状态下明确禁用“跳过”和“推迟”，防止用户认为点击失效。
- 开源首发应优先说明 Windows 支持完整，macOS/Linux 欢迎社区验证。

## 首发后优先级

1. 补充 macOS / Linux 构建验证。
2. 继续收口 DND、全局快捷键和系统权限提示。
3. 为开机自启、托盘菜单和多屏同步增加自动化测试或手工测试清单。
4. 增加无障碍说明，包括键盘操作、按钮 aria label 和高对比主题。
